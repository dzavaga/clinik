3M Бонд

Адгезив Single Bond Universal може застосовуватися для всіх технік, включаючи тотальне протруювання, самопротравліваніе селективне травлення. Він ідеально підходить як для прямих, так непрямих реставрацій може використовуватися на всіх поверхнях без окремого праймера.

C-Pilot Files

Файл C-Pilot (файл C-Pilot) Antaeos (Antaeos) складається з рукоятки, металевого стрижня з робочою частиною (секція стрижня, призначеного для виконання ендодонтичних маніпуляцій) і попередньо встановленої на штоку пробки. Рукоятка інструменту виконана з пластика, стрижень - з нержавіючої хромонікелевої сталі, пробка - з силікону. Неробоча частина штанги приладу градуйована рентгеноконтрастними насічками на відстані 18, 19, 20, 22 мм від верху для зручності визначення її довжини на рентгенограмі.

Прайм бонд NT

Адгезивна система є універсальним однокомпонентним праймером адгезивом в одному флаконі для забезпечення адгезії композитівкомпомерів до емалі дентину, також до металу кераміки. Прайм Бонд це адгезив 5 покоління для техніки тотального травлення, на основі розчинника ацетону.
